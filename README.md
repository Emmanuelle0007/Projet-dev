# 🏨 LuxeStay — Landing Page (Membre 1)

Projet : Réservation d'Hôtel | React + TypeScript + Tailwind CSS

## Installation

```bash
npm install
npm run dev
```

## Structure

```
src/
├── App.tsx
├── main.tsx
├── index.css
└── components/
    ├── Navbar.tsx       # Barre de navigation fixe
    ├── Hero.tsx         # Section principale avec recherche
    ├── Hotels.tsx       # Grille des hôtels disponibles
    ├── Offers.tsx       # Offres spéciales + newsletter
    ├── Testimonials.tsx # Avis clients + trust bar
    └── Footer.tsx       # Pied de page complet
```

## Branche Git

```bash
git checkout -b feature/landing-page
```

## Déploiement (Vercel)

```bash
npm run build
# Puis importer sur vercel.com
```
